No CSS nós possuímos os elementos por ID

  • Seletor por Tipo: Busca elementos por uma tag HTML especifica
  • Seletor por ID (#): Busca elementos através do atributo "id"
  • Seletor por classe (.): Busca elementos através do atributo "Class"
  • Seletor Universal (*): Seleciona todos os elementos HTML
  • Seletor de Atributo ([atrib]): Seleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos também buscar atributos com um valor especifico

Agrupamento de Seletores

Podemos aplicar as mesmas regras CSS para várias seletores diferentes. Para isso, podemos separar os seletores utilizando uma vírgula(,)

Assim, não precisamos ficar copiando e colocando os mesmos códigos para todos, nós podemos fazer um só bloco de CSS e reutilizar as mesmas regras!
